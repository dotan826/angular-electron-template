# AngularElectronTemplate

## What is it ?

I created a base template for an Electron Desktop App that include Angular as UI.

## Notes

* Typescript Support for both Electron & Angualr.
* Angular UI has base app UI component.
* Added live-watch for dev (both for electron and angular).

## Versions

* Electron -> 39.2.6
* Angular -> 19.2.6

## Instructions
1. Clone the project.
2. Go to "package.json" and change the "name" of the project.
3. Run "npm run dev".


Enjoy ! :)
